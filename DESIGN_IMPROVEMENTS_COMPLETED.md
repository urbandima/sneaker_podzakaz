### ✅ 5. Карточки товаров - оптимальный размер

**Проблема**: Карточки занимали слишком много места

**Требования пользователя**:
- **Desktop**: 5 колонок × 3 строки
- **Mobile**: 2 колонки × 3 строки

**Решение**: Создан класс `.grid-5` с адаптивной сеткой

**Файл**: `views/catalog/index.php`

**HTML**:
```html
<!-- БЫЛО: -->
<div class="products grid-4" id="products">

<!-- СТАЛО: -->
<div class="products grid-5" id="products">
```

**CSS**:
```css
/* Mobile: 2 колонки */
.products.grid-5 {
  grid-template-columns: repeat(2, 1fr);
  gap: 1rem;
}

/* Tablet: 3 колонки */
@media (min-width: 640px) {
  .products.grid-5 {
    grid-template-columns: repeat(3, 1fr);
    gap: 1.25rem;
  }
}

/* Desktop: 5 колонок ✅ */
@media (min-width: 1024px) {
  .products.grid-5 {
    grid-template-columns: repeat(5, 1fr);
    gap: 1.5rem;
  }
}
```

**Результат**:
- 📱 **Mobile (320-640px)**: 2 × 3 = 6 товаров на экран
- 💻 **Desktop (1024px+)**: 5 × 3 = 15 товаров на экран

**Эффект**: На 66% больше товаров видно на одном экране!
