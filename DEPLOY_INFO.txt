📝 КАК НАЙТИ ЭТИ ДАННЫЕ:

1. SSH данные:
   - Зайдите в cPanel
   - Найдите раздел "SSH Access" или "Terminal"
   - Там будут Username и Host
   - Пароль - тот же что для входа в cPanel

2. Путь к папке:
   - В cPanel → File Manager
   - Посмотрите структуру папок
   - Обычно: /home/ваш_username/public_html/

3. MySQL данные:
   - В cPanel → MySQL® Databases
   - Создайте новую БД если еще не создали
   - Создайте пользователя
   - Назначьте пользователя к БД (ALL PRIVILEGES)

4. GitHub:
   - Зайдите на github.com
   - Создайте новый репозиторий (Private)
   - Скопируйте URL
