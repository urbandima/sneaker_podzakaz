# Тестирование единой ширины контейнеров

## Быстрая проверка

### 1. Очистите кэш браузера
```bash
# Или используйте Ctrl+Shift+R / Cmd+Shift+R для жесткой перезагрузки
```

### 2. Откройте страницы в браузере

Проверьте следующие страницы на **одинаковую ширину контента**:

#### ✅ Главная страница (Header)
- URL: `/`
- Проверьте: Header navbar должен иметь ширину 1400px на больших экранах

#### ✅ Каталог
- URL: `/catalog`
- Проверьте: Контейнер каталога должен совпадать по ширине с header

#### ✅ Страница товара
- URL: `/catalog/product/[любой-товар]`
- Проверьте: Контейнер товара должен совпадать по ширине с header и каталогом

#### ✅ Админ-панель
- URL: `/admin/orders`
- Проверьте: Контейнер админки должен совпадать по ширине с остальными страницами

### 3. Визуальная проверка

**На экране > 1400px:**
- Все контейнеры должны иметь одинаковую ширину 1400px
- Контент должен быть выровнен по центру
- Одинаковые отступы слева и справа

**На экране < 1400px:**
- Контейнеры должны занимать 100% ширины с адаптивными отступами
- Отступы должны быть одинаковыми на всех страницах

### 4. Инструмент разработчика

Откройте DevTools (F12) и проверьте:

```css
.container {
    max-width: 1400px !important;
    margin-left: auto !important;
    margin-right: auto !important;
}
```

Этот стиль должен применяться ко всем контейнерам.

## Ожидаемый результат

✅ Header, Catalog и Product имеют одинаковую ширину  
✅ Нет визуальных "прыжков" при переходе между страницами  
✅ Контент выровнен по центру на всех страницах  
✅ Одинаковые отступы на всех разрешениях  

## Если что-то не работает

1. **Очистите кэш приложения:**
   ```bash
   rm -rf runtime/cache/*
   ```

2. **Проверьте загрузку CSS:**
   - Откройте DevTools → Network
   - Проверьте, что `container-system.css` загружается первым
   - Убедитесь, что нет ошибок 404

3. **Проверьте порядок CSS в AppAsset.php:**
   ```php
   public $css = [
       'css/container-system.css', // Должен быть первым!
       'css/site.css',
       // ...
   ];
   ```

## Изменение ширины (если нужно)

Отредактируйте `/web/css/container-system.css`:

```css
:root {
    --container-max-width: 1400px; /* Измените на нужное значение */
}
```

Популярные варианты:
- **1200px** — компактный дизайн
- **1400px** — текущее значение (оптимально для e-commerce)
- **1600px** — широкий дизайн
